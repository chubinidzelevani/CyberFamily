Security & Encryption Model
🔑

User Password

Hashed with Argon2 + salt. Never stored in plaintext.

📝

BIP-39 Mnemonic (24 words)

Required to trust a new device. Derives the Master Encryption Key (MEK).

🧠

Master Encryption Key (MEK)

Cached per trusted session. Never stored on server. All data enc/dec requires MEK.

🔒

AES-256-GCM

Applied to all sensitive data. Each field has unique nonce + ciphertext + auth tag.

🌐

JWT Bearer Tokens

Token hash stored in DB — never plaintext. Session invalidated on logout.

🛡

TLS / SSL

Real cert for cyberfamily.ge — valid until Feb 2027. HTTP redirects to HTTPS.

✔ Encrypted at Rest
